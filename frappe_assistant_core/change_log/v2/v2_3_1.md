## Version 2.3.1

### Database-Backed Plugin & Tool Management

- **FAC Plugin Configuration** DocType — atomic, database-backed plugin enable/disable with audit trail and multi-worker cache invalidation
- **FAC Tool Configuration** DocType — per-tool enable/disable, category assignment (read_only, write, privileged), and role-based access control
- **Automatic tool category detection** via AST parsing of tool source code
- **Frappe V16 support** — official V16 compatibility alongside V15
- **No more race conditions** — database locks replace file-based state management
