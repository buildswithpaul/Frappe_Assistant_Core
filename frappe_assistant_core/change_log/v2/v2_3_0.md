## Version 2.3.0

### MCP Prompt Templates

- **Prompt Templates system** — create reusable, parameterized prompts that users can customize and execute via AI clients
- **MCP prompts protocol** support (`prompts/list`, `prompts/get`)
- **Template rendering** with Jinja2, Format String, and Raw engines
- **Access control** — Private, Shared, or Public visibility with role-based sharing
- **7 built-in templates** — Sales analysis, Manufacturing, HR, CRM, Procurement, Documentation, Data quality
- **Dark theme support** for FAC Admin and Assistant Core Settings pages
- **OAuth timezone fix** — tokens no longer expire prematurely in non-UTC timezones
